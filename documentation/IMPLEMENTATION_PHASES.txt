PHASE 10 - Bounding Box Overlap Engine
====================================================================
FILES: ai-core/agents/vision_agent.py | ai-core/tools/playwright_tool.py
WHAT:
  - PlaywrightTool: evaluate JS to get getBoundingClientRect() for all
    text + interactive elements, return list of bounding boxes
